How it works

  1. Configure the S3 repository here.
  2. Enable DR on individual nodes from their detail page.
  3. The server pushes credentials to enabled nodes via heartbeat responses.
  4. Nodes back up their configuration to the S3 repo on the configured interval.
  5. The DR shield on each node shows the current backup status.
