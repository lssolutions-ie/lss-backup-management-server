Anomaly Retention

Controls how long acknowledged anomalies stay on the live /anomalies page before moving to the archive.

After this many days, acknowledged anomalies disappear from /anomalies but remain forever on /anomalies/archive. Default 30.

Anomaly Detection Thresholds

Triggers when a backup run shows unexpected drops vs. the previous run — possible signs of source deletion, ransomware, or repo tampering. Each anomaly is logged for forensics and visible on the Anomalies page.

Alert when the restic repo loses more than this many snapshots between two reports. Default 1.
